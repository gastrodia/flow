# DataFlow项目简介

### Why

1. 项目间数据结构不统一 （家具、家装、平台）

2. 前后端数据结构不统一 （Java、WebGL、前端）

3. 数据管理方式原始、开发成本高（Admin界面？）

4. 公司内部数据分散、使用方式单一（产品、运营数据）

5. 公司外部数据没有好的汇集和使用机制（复博士）
